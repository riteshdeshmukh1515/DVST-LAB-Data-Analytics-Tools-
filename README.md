
# 📊 DVST Lab – Data Visualization and Storytelling

## 🧾 Overview

This repository contains practical assignments and dashboard projects developed as part of the **Data Visualization and Storytelling (DVST) Lab**. The main objective is to explore datasets, perform data analysis, and present insights through interactive dashboards using modern BI tools.

The projects focus on transforming raw data into meaningful visual stories that help in better decision-making.

---

## 🛠️ Tools & Technologies Used

* 📊 Microsoft Power BI
* 📈 Tableau
* 📂 Excel / CSV datasets
* 🧠 Data Analytics & Visualization Techniques
* 📉 Dashboard Design & Storytelling Concepts

---

## 📁 Datasets Used

The following datasets were used to build dashboards and analyze insights:

* 🦠 COVID-19 Dataset
* 🚗 Car Dataset
* 🎓 Student Performance Dataset
* 🏢 Placement Dataset
* ❤️ Heart Disease Dataset

---

## 📊 Project Objectives

* Clean and preprocess raw datasets
* Perform exploratory data analysis (EDA)
* Identify patterns, trends, and correlations
* Create interactive dashboards
* Present insights using visual storytelling techniques

---

## 📌 Dashboards Developed

### 1. COVID-19 Dashboard

* Global and regional case trends
* Recovery and death rate analysis
* Time-series visualization of cases

### 2. Car Dataset Dashboard

* Price distribution analysis
* Fuel type and brand comparison
* Mileage and performance insights

### 3. Student Performance Dashboard

* Academic score distribution
* Gender-wise and subject-wise analysis
* Performance trends and factors

### 4. Placement Dashboard

* Placement status analysis
* Salary distribution
* Skill vs placement correlation

### 5. Heart Disease Dashboard

* Risk factor analysis
* Age and gender distribution
* Medical attribute correlations

---

## 📈 Key Learnings

* Data cleaning and transformation techniques
* Dashboard designing principles
* Storytelling with data visualizations
* Hands-on experience with BI tools
* Analytical thinking and insight generation

---

## 🎯 Outcome

This lab helped in understanding how raw datasets can be converted into meaningful insights using visualization tools like Power BI and Tableau, improving decision-making through data storytelling.
